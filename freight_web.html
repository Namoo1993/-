🔗 Supabase 연결 설정

팀원과 데이터를 공유하려면 Supabase 프로젝트 정보가 필요합니다.
아래 안내를 따라 무료로 설정하세요. 한 번만 입력하면 저장됩니다.

1supabase.com → 무료 가입 → New Project 생성
2Project Settings → API 탭 클릭
3Project URL과 anon public key를 아래에 붙여넣기
4SQL Editor에서 아래 SQL 실행: 
CREATE TABLE templates (
  id text PRIMARY KEY,
  name text NOT NULL,
  rows jsonb NOT NULL,
  updated_at timestamptz DEFAULT now()
);
CREATE TABLE estimates (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  title text NOT NULL,
  note text,
  cnt20 int DEFAULT 0,
  cnt40 int DEFAULT 0,
  bl int DEFAULT 1,
  rows jsonb NOT NULL,
  created_by text,
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);
ALTER TABLE templates ENABLE ROW LEVEL SECURITY;
ALTER TABLE estimates ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public_all" ON templates FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "public_all" ON estimates FOR ALL USING (true) WITH CHECK (true);
